# 🚀 StockVault

## 🌟 Overview

**StockVault** is a full-stack stock market web application that allows users to track stocks, manage portfolios, and view market insights through an interactive dashboard.

> 💡 Built using the MERN stack with real-world API integration.

---

## ✨ Features

✔️ User Authentication (Login / Signup)
✔️ Live Stock Data Integration
✔️ Interactive Dashboard UI
✔️ Portfolio Management System
✔️ Secure Backend APIs
✔️ Responsive Design (Mobile + Desktop)

---

## 🧠 Tech Stack

| Layer    | Technology                |
| -------- | ------------------------- |
| Frontend | React.js, Tailwind CSS    |
| Backend  | Node.js, Express.js       |
| Database | MongoDB                   |
| API      | (Finnhub / Alpha Vantage) |

## ⚙️ Setup Instructions

### 🔧 Clone Repository

```bash
git clone https://github.com/your-username/stockvault2.git
cd stockvault2
```

---

### 📦 Install Dependencies

```bash
cd backend && npm install
cd ../frontend && npm install
```

---

### 🔐 Environment Variables

Create a `.env` file inside **backend/**:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
API_KEY=your_stock_api_key
JWT_SECRET=your_secret_key
```

---

### ▶️ Run Application

```bash
# Start backend
cd backend
npm start

# Start frontend
cd frontend
npm start
```
