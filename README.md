# 📈 StockVault — Virtual Trading Platform

A feature-rich **full-stack virtual stock trading platform** built for learning 
and practicing Indian stock market trading without any real money risk.

## ✨ Features

- 💰 ₹1,00,000 virtual balance on signup
- 📊 Real-time NSE/BSE stock prices via WebSocket
- 🕯️ Candlestick charts with zoom/pan (TradingView style)
- 🔄 F&O Options Chain — CE/PE with MIS/NRML
- 🚀 IPO Centre — Apply, track & view listed IPOs
- 💼 Portfolio with live P&L tracking
- 📋 Complete order history & P&L report
- 💰 SIP & Mutual Fund calculator
- ⭐ Watchlist with price alerts
- 🔔 Notification system
- 🎁 Referral & Earn system (₹599/referral)
- 🌙 Dark/Light theme
- 📱 Mobile responsive

## 🛠️ Tech Stack

**Frontend:** React.js, Chart.js, Lightweight Charts, CSS Modules  
**Backend:** Node.js, Express.js, MongoDB, WebSocket  
**Auth:** JWT Authentication  
**Payments:** Razorpay integration
